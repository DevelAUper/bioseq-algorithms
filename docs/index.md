# Documentation Index

This index maps each document in `docs/` to its purpose and current scope.

- `docs/getting-started.md`: prerequisites, build/test commands, and first runs for CLI and GUI.
- `docs/cli.md`: complete command reference for `global_linear`, `global_count`, and `global_affine`.
- `docs/fasta-and-io.md`: FASTA parsing behavior, sequence handling, and I/O conventions.
- `docs/scoring-matrices.md`: Phylip-like matrix format, validation rules, and lookup behavior.
- `docs/pairwise-global-linear.md`: linear-gap global alignment algorithm and traceback model.
- `docs/counting-optimal-alignments.md`: counting co-optimal alignments for linear and affine models.
- `docs/parallelism-analysis.md`: wavefront anti-diagonal parallelism design and benchmark findings.
- `docs/experiments-runtime.md`: runtime experiment setup and reporting guidance.
- `docs/glossary.md`: short definitions of biology and algorithm terms used in the project.

For a top-level project overview, see `README.md`.
